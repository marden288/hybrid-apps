Technical Specifications

⚡ Power System

  • Total Power: 1.2 kW (600W Solar + 600W Wind)
  • Battery: 12V 200Ah Deep Cycle (×2)
  • Charge Controller: MPPT 60A + PWM 30A
  • Inverter: 1500W Pure Sine Wave

🔒 Security System

  • Authentication: 4-digit PIN via keypad
  • Alerts: SMS via SIM800L GSM
  • Lock: 12V Solenoid, 10kg force
  • Enclosure: Tamper-proof steel

🔌 Charging Ports

  • USB-A: 4 ports × 5V / 2.4A
  • USB-C PD: 2 ports × 65W
  • DC: 12V outlet
  • AC: 220V via inverter

🤖 Controller

  • MCU: Arduino Nano (ATmega328P)
  • Display: 16×2 LCD with I2C
  • Input: 4×4 Matrix Keypad
  • Communication: GSM (SIM800L)
